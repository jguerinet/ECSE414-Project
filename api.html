Registering as a Peer

Use the following HTTP requests to register as a peer:

  • GET /peers/ -> Returns the list of registered peers.
  • GET /peers/%peer_id -> Returns specific peer.
  • POST /peers/ -> Adds a new peer to table. The peer's properties are sent as JSON in the request body.
  • DELETE /peers/%peer_id -> Returns specific peer.

Registering a Call

Use the following HTTP requests to initiate a call with another peer:

  • GET /calls/ -> Returns the list of existing calls.
  • GET /calls/%peer_id -> Returns any peers that have added calls to the call table for which peer_id is the receiver. Queried by client when client is idle.
  • POST /calls/ -> Adds a new call to the call table. Call properites are sent as JSON in the request body.
  • DELETE /calls/%peer_id -> Removes any calls from the calltable for which peer_id is the receiver
